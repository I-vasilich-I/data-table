# Data-Table

## How to run the project

### Clone repository

```bash
git clone https://github.com/I-vasilich-I/data-table.git
```

Go to `data-table` folder

```bash
cd ./data-table
```

### Installation

```bash
npm install
```

### Running project

  ```bash
  npm run start
  ```

### Running project with Docker

  ```bash
  npm run docker
  ```
